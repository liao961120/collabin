為什麼夜深了還不回家？

雨下得唏哩嘩啦，風吹得窗戶哐哐響，屋頂上不斷地大珠小珠落玉盤…這種時刻~~城市裡的每一位路人都想快點回家。
路人把自己塞進被窩裡，取暖；閒得發慌就開始每分無聊時刻的日常，滑滑手機、追追劇；餓了可以勉強抱著一團溫暖煮泡麵去，在迷濛的香氣與蒸氣中祭一下五臟廟；累了就乖乖閉眼睡去，睜眼又是嶄新的一天！
像這樣直接啟動休假模式，人生快意得很～ 不過…

這路人不走在一般道路上，正坐在一處聽風聽雨的小地方，思考著資料科學的問題。
這路人不是往家裡鑽，走上一條從來沒有想過的嘗試，問了自己一個關於怎麼寫一個資料科學的小品文章的問題。

於是乎，這路人深夜了還不回家。
他心裏暗暗下了一句：「今天不寫出來，明天根本寫不出來。」 恰恰與我(們)一模模一樣樣。

PS: 我還是寫到半夜啊…

下筆之前先做功課

從各種端不上檯面的人生經驗與教訓來看，可以的話「先做功課」會是很好的。

由於不知道究竟要寫什麼，下筆下得很痛苦，有一個大的概念還是覺得自己的點子弱爆，更下不了筆、動不了手、寫不出code。如果蟹老闆說的是真的，那…至少我還可能可以寫文章，小品的那種。那麼小品文到底是什麼？ （下文引自維基百科）

小品文是種散文的流派，其內容經常是在生活中，作者經過深思而傳達給讀者的訊息，內容題材不限，所傳達的道理或想法也沒有限制。其特色是以深入淺出、簡明生動的筆法，描述或議論一些事理。

讀完立刻覺得這不適合我。

特點主要是通過事實和藝術形象來表現思想內容，議論較少，敘述、描寫或抒情成分較多，以幽默的方式和喜劇性的情節，活潑、輕鬆的語言，給人一種揭露性的笑，並使人在笑過之後，看到問題的實質，發人深思。

但是，少議論、描述或抒情成分較多、輕鬆、活潑、笑，這些倒是我可以辦到的。

開一小道 *

想了老半天，想了一長串，不如就動手試試看。每次踩一小步，就算很醜，一點一滴就會變成小小的一大步！
大當家跟掌櫃的都沒有給跑堂的設大限了，只要跟資料科學、語言學相關而且有寫到 code 就可以是嗎？
那我可要寫好玩的，會看到自己進步與努力的東西，自己開一條小道去未來。
同時我也很期待大家會開出什麼 ：）

在我生命中，現在的大環境有著比過去二十年更多的資料，資料也相對更容易取得與分析。我覺得 Sean 上一堂的導論分享很有道理，得到很多資料確實有很多可以玩的，但是也很容易沉到資料的大海底、落到資料分析的大缽中。

要去哪裡呢？

雖然統計可能無法完美解決一切疑問，但作為一門發展長久且關心由樣本資料合宜推估到完美(假想)母群的一門學科，它的思維與作法是值得瞭解與實際嘗試的。或許我們能有的資料總有分析、詮釋的侷限，但我們解決疑問的方法、思考的問題與好奇的想像可以是無限的！（我覺得人總是有辦法去觸類旁通點什麼的）

觀察擁有的資源與目標

放眼可觸及的資源與資料中，我覺得最能引起我興趣的是跟語奧有關的資料。因為兼差工作的關係，加上國際教育背景的影響，這類的資料總是能吸引我。來到語言所，在學習 R 與 Python 等相關知識與技能之後，我深深覺得掌握資料可以掌握很多事情。而且不只文字資料，連以前只想離得遠遠的數據資料也越來越有魔力了。因此，我這第一篇文章就這麼跟語奧掛勾啦！

提出覺得有趣的、關心的問題

今年語奧的報名人數有了翻倍的提升，2016 年最剛開始的時候已有 86 人到考，但我從沒想過到考人數會突破 200 。當年那個很少人知道的競賽，現在是不是有越來越多人知道了呢？還是只是在語言學自己的領域與同溫層中發酵著？真的有更多人對語言學感興趣嗎？能夠再做些什麼來提供更多人、更多的接觸機會嗎？這一切是怎麼發生的？…問題好像可以一直不斷地冒出來。在這麼多的問題中，我最想知道： 這些年來大家是否對語言學的了解多更多了呢？

解決提問的方法之一

扣著這個問題，配上手邊可以找到的資料，我覺得可以由「對語言學的瞭解分享」看語言學被理解的樣貌。
由於 coding 需要複習一下，所以我就先做一個小的練習：讀取文字資料並製作詞頻表，之後會慢慢加入更多東西。

可以進步的第一小步

首先，恭喜我自己終於放下心防使用了 lapply ！感覺到可以寫出簡潔的 code 是非常舒服的一件事情！值得努力。接下來是自己覺得可以進步的地方：

  1. 可以把詞頻表再清一下
  2. 可以分年呈現

第二小步就是試圖對這些結果做比較，看看個年度的分享內容差異等。或許可以有些自己想的假說，再用資料做驗證。

花開有其時

其實從構思主題到實作而後書寫真的很磨人，真的是一個好的鍛鍊。過程中的思考、認識自己的脾性、放下完美的堅持與維持興趣、用三腳貓功夫去實作、拆解每一個過程與步驟、練習搭建合理的推論與架構…等，好多挫折、好多收穫、好多成就。
沒有很難，只是有點挑戰。一定不會很容易，但是依然可以去努力。
在環境存在支持足夠的時候，花開有其時，碩果必可期。
